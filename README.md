# AI Study Companion – Capstone Project

## Overview
AI Study Companion is a mobile application developed as a capstone project.  
The application is designed to help students organize their study process by combining traditional study tools with AI-powered assistance.

The system follows a client–server architecture with a Flutter mobile frontend and a PHP backend.

## Features
- User authentication (Login & Register)
- Notes management (add, view, delete)
- AI-generated summaries for notes
- Flashcards system
- Pomodoro timer for focused study sessions
- Task management and progress tracking
- Dashboard showing study progress

## Technologies Used

### Mobile Application
- Flutter (Dart)
- Android Studio

### Backend
- PHP (RESTful APIs)
- MySQL database
- XAMPP (local server)

### AI Integration
- OpenRouter API for AI-powered summaries

## Project Structure
AI-Study-Companion/
├── app/        # Flutter mobile application
├── hosting/    # PHP backend APIs
└── README.md

## How It Works
1. Users create an account or log in.
2. Notes are created and stored in the database.
3. AI summaries are generated upon request using the OpenRouter API.
4. Users can manage flashcards, tasks, and Pomodoro sessions.
5. Progress is tracked and displayed inside the app.
